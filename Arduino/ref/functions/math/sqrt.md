# sqrt()

ここは`sqrt()` 関数のページです

# 説明

平方根を計算します

# 構文

`sqrt(x)`

# パラメータ

`x`：平方根を計算したい値。すべてのデータ型が使えます

# 返り値

$ \sqrt x $
の計算結果。データ型は`double`
