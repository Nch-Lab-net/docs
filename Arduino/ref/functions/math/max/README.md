# max()

ここは`max()` 関数のページです

# 説明

2つの値の最大値を計算します

# 構文

`max(x, y)`

# パラメータ

`x`：1つ目の数字。すべてのデータ型が使えます
`y`：2つ目の数字。すべてのデータ型が使えます

# 返り値

2数のうち値が大きい方

# サンプルコード

このコードでは`sensVal`の値が少なくとも20以上となるようにしています

```cpp
sensVal = max(sensVal, 20); // sensVal と 20 を指定
                            // 少なくとも 20 以上になります
```

# 注意点

括弧内で他の関数を実行すると予期しない結果になることがあります

```cpp
max(a--, 0);  // 不正な結果とならない様、このコードは避けてください

// 代わりに以下のコードを使用してください:
max(a, 0);
a--;  // 他の演算を括弧の外で行う
```
