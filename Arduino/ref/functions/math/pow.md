# pow()

ここは`pow()` 関数のページです

# 説明

数値のべき乗を計算します

# 構文

`pow(base, exponent)`

# パラメータ

`base`：底となる数字。データ型は`float`  
`exponent`：指数。データ型は`float`  

# 返り値

$ base^{exponent} $
の計算結果。データ型は`double`

# サンプルコード

$ x^y $
を計算します

```cpp
z = pow(x, y);
```
