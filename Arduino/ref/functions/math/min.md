# min()

ここは`min()` 関数のページです

# 説明

2つの値の最小値を計算します

# 構文

`min(x, y)`

# パラメータ

`x`：1つ目の数字。すべてのデータ型が使えます  
`y`：2つ目の数字。すべてのデータ型が使えます  

# 返り値

2数のうち値が小さい方

# サンプルコード

このコードでは`sensVal`の値が100を超えることがないようにしています

```cpp
sensVal = min(sensVal, 100);  // sensVal と 100 を指定
                              // 少なくとも 100 以下になります
```

# 注意点

括弧内で他の関数を実行すると予期しない結果になることがあります

```cpp
min(a++, 0);  // 不正な結果とならない様、このコードは避けてください

// 代わりに以下のコードを使用してください:
min(a, 0);
a++;  // 他の演算を括弧の外で行う
```
